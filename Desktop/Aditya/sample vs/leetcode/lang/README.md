# RAG_DEMO

A learning project for Retrieval-Augmented Generation (RAG) systems.

## Project Structure

- `adi1/` - Main project directory containing RAG learning materials
  - `rag_learning_notebook.ipynb` - Jupyter notebook with RAG learning examples
- `chat1.ipynb` - Chat-related notebook
- `add1.py`, `mul.py`, `hello.py` - Python utility scripts
- `.env` - Environment configuration

## Getting Started

1. Install required dependencies
2. Explore the notebooks in the `adi1/` directory
3. Review Python scripts for utility functions

## Requirements

- Python 3.x
- Jupyter Notebook
- See `.env` for environment configuration

## Usage

Run the Jupyter notebooks to learn about RAG implementation:
```bash
jupyter notebook adi1/rag_learning_notebook.ipynb
```
